# appdev1-sdg4-education
Our group selected SDG 4: Quality Education.
 - SDG 4 focuses on ensuring inclusive and equitable quality education and promoting lifelong learning opportunities for all.

Rationale:

- We chose SDG 4 because education is very important in improving people's lives and future opportunities. Many students still face challenges such as lack of access to learning resources, poor internet connection, and limited educational support.
As students, we believe we can help by creating a web application that supports learning, provides useful information, or makes education more accessible and engaging.
